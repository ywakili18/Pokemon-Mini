Hi and welcome to my personal project: Pokemon Gold Mini! This game is inspired by one of my favorite games ever : Pokemon Gold for the GameBoy Color!

Pokemon Gold Mini utilizes basic HTML, CSS, and Javascript.

This game is currently in pre-alpha stage. Future versions will contain different pokemon with a variety of movesets and sprites.

So what does this version contain?

Player 1 picks the first Pokemon: in this version it is only Typhlosion or Charizard. Select the move corresponding with each Pokemon and let the damage roll in! When HP is at 0 or lower a winner will then be declared.
